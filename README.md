# IoT Laboratory Activities

## Overview

This repository contains a collection of IoT laboratory activities implemented using Arduino, ESP32, STM32, Raspberry Pi Pico, and TinyML. Each activity demonstrates the implementation of a real-world IoT application through simulation using platforms such as Tinkercad and Wokwi. The projects focus on automation, environmental monitoring, and smart system development.

## Technologies Used

* Arduino Uno
* ESP32
* STM32
* Raspberry Pi Pico
* TinyML
* Tinkercad
* Wokwi Simulator
* C/C++
* MicroPython

## Project Activities

## Activity 1 – Smart Fire Alarm System using Arduino

### Description

A Smart Fire Alarm System developed using Arduino Uno in Tinkercad to simulate early fire detection. The system continuously monitors the temperature and triggers visual and audible alerts when the temperature exceeds a predefined threshold, enabling quick detection of potential fire hazards.

### Features

* Real-time temperature monitoring
* Automatic fire detection based on threshold value
* LED indication for fire alerts
* Buzzer alarm for immediate notification

## Activity 2 – Smart Temperature Monitoring System using ESP32

### Description

A Smart Temperature Monitoring System developed using ESP32 in Wokwi to simulate continuous temperature monitoring. The system measures temperature in real time and generates alerts whenever the temperature exceeds a predefined threshold, helping to identify unsafe environmental conditions.

### Features

* Real-time temperature monitoring
* Threshold-based temperature alerts
* LED/Buzzer indication for high temperature
* Serial Monitor output for live readings

### Simulation Link

https://wokwi.com/projects/460845442125174785


## Activity 3 – Smart Irrigation System using STM32

### Description

A Smart Irrigation System developed using STM32 in Wokwi to automate irrigation based on soil moisture levels. The system continuously monitors soil moisture and activates irrigation when the moisture falls below a predefined threshold, ensuring efficient water usage and healthier crop growth.

### Features

* Real-time soil moisture monitoring
* Automatic irrigation control
* Threshold-based water pump activation
* Efficient water management

### Simulation Link

https://wokwi.com/projects/460846540843282433


## Activity 4 – Automatic Light Control System using Raspberry Pi Pico

### Description

A Smart Automatic Light Control System developed using Raspberry Pi Pico in the Wokwi simulator to automate lighting based on ambient light intensity. The system continuously monitors light levels using an LDR sensor and automatically switches the light ON in dark conditions and OFF in bright conditions, improving energy efficiency.

### Features

* Real-time ambient light monitoring
* Automatic light control using an LDR
* Energy-efficient lighting automation
* Serial Monitor output for live light intensity readings

### Simulation Link

https://wokwi.com/projects/460850902742046721


## Activity 5 – TinyML-Based Clap Detection System

### Description

A TinyML-based Clap Detection System developed in the Wokwi simulator to enable touch-free control using sound recognition. The system detects clap sounds and performs an action, demonstrating how TinyML can be used for efficient human-machine interaction in smart environments.

### Features

* Clap sound detection using TinyML
* Touch-free device control
* Real-time sound classification
* Serial Monitor output for detection status

### Simulation Link

https://wokwi.com/projects/460938409570791425
